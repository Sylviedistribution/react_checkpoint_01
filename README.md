# React Checkpoint 1 - Product Card

This project was created using **Create React App** and **React Bootstrap**.  
It displays a product card showing the product's name, image, description, and price.

## Installation

```bash
npm install
npm start
```

## Files Structure

```
src/
 ├── components/
 │    ├── Name.js
 │    ├── Description.js
 │    ├── Price.js
 │    ├── Image.js
 │    ├── Greetings.js
 │    ├── Header.js
 │    └── Footer.js
 ├── product.js
 ├── App.js
 └── index.js
```

## Description

- `product.js`: Contains a product JSON object.
- `Name.js`, `Description.js`, `Price.js`, `Image.js`: Display product info.
- `Greetings.js`: Shows a message depending on user's name.
- `Header.js` & `Footer.js`: Basic layout components.
- `App.js`: Main file assembling everything.

## Commands

- `npm install`: Install dependencies.
- `npm start`: Run the app.
